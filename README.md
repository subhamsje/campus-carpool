## 🛠️ Tech Stack: The Engine Under The Hood

<div align="center">
  <img src="https://skillicons.dev/icons?i=react,ts,nodejs,express,prisma,sqlite,supabase,vite,tailwind,jest" />
</div>

<br/>

<details>
<summary><b>📐 System Architecture</b></summary>
<p>
The project follows a modular, clean-architecture approach:
- <b>Frontend:</b> Vite + React + TypeScript with centralized state.
- <b>Backend:</b> Node.js Express with modular routing.
- <b>Database:</b> Prisma ORM for type-safe database access.
- <b>Realtime:</b> Supabase for instant data synchronization.
</p>
</details>

## 🏎️ Getting Started

```bash
# Clone the repository
git clone https://github.com/subhamsje/campus-carpool.git

# Enter the cockpit
cd campus-carpool

# Fuel up (Install dependencies)
npm install

# Ignite the engine (Start dev server)
npm run dev
```
