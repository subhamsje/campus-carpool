### 🛰️ Mission Control

**Campus Carpool** is a high-octane mobility layer for students. Built with a focus on **Zero-Latency Matching** and **Gamified Trust**, it transforms the daily commute into a social, efficient, and sustainable experience.

[Explore Docs](https://github.com/subhamsje/campus-carpool) • [Report Bug](https://github.com/subhamsje/campus-carpool/issues) • [Request Feature](https://github.com/subhamsje/campus-carpool/issues)

## 🏁 Ignite Local Environment

```bash
# 1. Clone the repository
git clone https://github.com/subhamsje/campus-carpool.git

# 2. Enter the cockpit
cd campus-carpool

# 3. Fuel up (Install dependencies)
npm install

# 4. Ignite the engine
npm run dev
```
